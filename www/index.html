Android on Linux, managed.

waydroid-toolkit is an open-source CLI and Python library for managing Waydroid — Android-in-a-container on Linux via Incus or LXC.

Get started View on GitHub
pip install waydroid-toolkit

What wdt does

Container management

Start, stop, snapshot, export, and import the Waydroid container. Supports both Incus and native LXC backends.

Extensions

Install GApps, Magisk, ARM translation, microG, Widevine, and keymapper with a single command.

Images

Manage Waydroid image profiles, OTA updates, and AndroidTV images.

Streaming

Mirror the Waydroid display via scrcpy over ADB. Record sessions, forward audio, and control from the host. Ported from anbox-streaming-sdk.

Shared storage

Mount NFS shares and local directories into the Waydroid container via incus config device. Ported from anbox-cloud-nfs-operator.

CI / GitHub Actions

Run Android UI screenshot tests and E2E instrumented tests on GitHub runners using Waydroid as the Android backend. Ported from anbox-cloud-github-action and anbox-cloud-demos.

Terraform

Provision Incus containers running Waydroid with the incus Terraform provider. Ported from anbox-cloud-terraform.

Packages & ADB

Install and remove APKs, manage F-Droid repos, run ADB commands, and forward USB devices into the container.

Quick start

# Install
pip install waydroid-toolkit

# Check status
wdt status

# Install GApps
wdt extensions install gapps

# Mirror display
wdt stream start

# Mount NFS share
wdt storage nfs add 192.168.1.10:/exports/assets

# Snapshot
wdt container snapshot create before-update
